Create an event

To create an event in Whiti, navigate to the 'Create Event' page.

Enter in an optional event title and description that will be displayed on the event page, then press 'Create Event'.

You will be taken to a new page which displays your unique event code. Make sure to share this with your users so that they can join the event.

Join an event

Your first action is to choose whether you are joining the event as a listener or an intepreter. If you are wanting to hear the interpretation then join as a 'Listener'. If you will be interpreting the speaker at the event, then pick 'Interpreter'.

You should now be taken to a page that prompts you to enter an event code. One should be provided at the event, please enter this into the provided text field and press 'Join'

If you joined the event as an interpreter, you will now be able to broadcast your interpretation to the listeners by speaking into your device's mic.

If you joined the event as a listener, you will now be able to receive audio from the interpreter.
